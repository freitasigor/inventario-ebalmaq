# Sistema de Inventário Ebalmaq 📦

Um sistema web responsivo desenvolvido para gestão e controle de inventário de patrimônios e estoque. Este projeto foi estruturado para ser leve, ágil e de fácil navegação.

## 💻 Tecnologias Utilizadas
* **Front-end:** HTML5, CSS3, Bootstrap 5
* **Back-end:** Node.js
* **Banco de Dados:** SQLite (arquivos `.db` locais)
* **Manipulação de Dados:** Leitura e importação de planilhas CSV

## ⚙️ Principais Funcionalidades
* Interface responsiva adaptada para desktops e dispositivos móveis.
* Gestão e visualização de estoque atualizado.
* Controle de entrada e saída de patrimônios.
* Sistema de importação de dados em massa via arquivos CSV.
* Identificação automática de patrimônios duplicados.

## 🚀 Como rodar o projeto localmente

Siga os passos abaixo para testar o projeto na sua máquina:

1. Clone este repositório:
```bash
git clone https://github.com/freitasigor/inventario-ebalmaq.git
```

2. Acesse a pasta do projeto no seu terminal:
```bash
cd inventario-ebalmaq
```

3. Instale as dependências necessárias do Node.js:
```bash
npm install
```

4. Inicie o servidor local:
```bash
node server.js
```

5. Abra o seu navegador e acesse a porta configurada (ex: `http://localhost:3000`).

## 🔒 Segurança e Privacidade (LGPD)
Este repositório serve como uma demonstração de portfólio. Todos os dados sensíveis, credenciais reais e informações corporativas foram removidos. O sistema opera exclusivamente com dados genéricos de teste (*mock data*) para garantir total conformidade com as boas práticas de Segurança da Informação e LGPD.
